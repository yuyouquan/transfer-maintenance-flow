## 1. 测试概述

### 1.1 测试范围

覆盖转维电子流系统的所有前端功能，包括：
- **工作台模块**：转维申请、记录列表、待办任务、电子流详情、资料录入与AI检查、维护审核、关闭流水线
- **配置中心模块**：转维材料模板配置、评审要素模板配置、版本管理

### 1.2 测试策略

- 前期使用Mock数据进行前端功能测试
- 重点关注：流程状态流转、权限按钮显示条件、数据联动、表格操作
- 按功能模块分组，每个原始需求点至少1个测试用例

### 1.3 测试环境

| 项目 | 配置 |
|------|------|
| 浏览器 | Chrome 120+, Safari 17+ |
| 分辨率 | 1920x1080, 1440x900 |
| 框架 | Next.js 16 + Ant Design 5 |
| 数据 | Mock数据 |

### 1.4 前置条件

- 系统已部署并可访问
- 存在Mock用户数据（张三/SPM，李四/TPM，王五/SQA等）
- 存在Mock项目数据和转维申请数据
